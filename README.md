# antisocial
A Stardew Mod where interacting with people takes energy


## Parameters
- EnergyCost (int2): amount of energy spent for each interaction. default 2
- DisableForEvents (bool): whether to lose energy in events. Note that any new dialogue box from another character will decrease energy. default false
- CanPassOut (bool): whether the player can pass out from socialization interactions. If false, energy will not decrease if it would go below -15. default false
